## 📌 About This Repository

This repository contains the Data Science internship tasks completed as part of the **AICTE Oasis Infobyte Internship Program**.

Each project is implemented using a Jupyter Notebook and follows a practical machine learning workflow, including data loading, preprocessing, exploratory data analysis, visualization, model training, evaluation, and prediction.

The repository includes projects from multiple areas of data science such as:

- Classification
- Regression
- Exploratory Data Analysis
- Natural Language Processing
- Sales and business prediction

## 🌸 Task 1: Iris Flower Classification

The Iris Flower Classification project predicts the species of an Iris flower based on its measurements.

### Key Highlights

- Uses the classic Iris dataset from `scikit-learn`
- Classifies flowers into Setosa, Versicolor, and Virginica
- Performs data visualization and correlation analysis
- Trains multiple classification models
- Compares model performance
- Demonstrates sample predictions

### Models Used

- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier
- Support Vector Machine
- K-Nearest Neighbors
- Gradient Boosting Classifier

---

## 📊 Task 2: Unemployment Analysis With Python

This project analyzes unemployment trends in India using Python.

### Key Highlights

- Loads and cleans unemployment data
- Analyzes unemployment trends by region and time
- Studies the impact of COVID-19 on unemployment
- Compares urban and rural unemployment patterns
- Creates meaningful visualizations using Matplotlib and Seaborn

### Analysis Includes

- Regional unemployment comparison
- Monthly trend analysis
- Labour participation analysis
- COVID-19 impact study
- Urban vs rural comparison

## 🚗 Task 3: Car Price Prediction

The Car Price Prediction project predicts the selling price of used cars using machine learning regression models.

### Key Highlights

- Loads car price datasets when available
- Generates sample data if no dataset is found
- Performs feature engineering
- Handles categorical and numerical features
- Trains multiple regression models
- Evaluates models using regression metrics

### Models Used

- Linear Regression
- Ridge Regression
- Lasso Regression
- Decision Tree Regressor
- Random Forest Regressor
- Gradient Boosting Regressor

## 📧 Task 4: Email Spam Detection With Machine Learning

This project builds a machine learning model to classify messages as spam or ham.

### Key Highlights

- Uses text preprocessing techniques
- Converts text into numerical features using TF-IDF
- Trains multiple classification models
- Evaluates models using accuracy, precision, recall, and F1 score
- Demonstrates spam prediction on sample messages

### Models Used

- Multinomial Naive Bayes
- Bernoulli Naive Bayes
- Logistic Regression
- Linear Support Vector Machine
- Random Forest Classifier

## 📦 Installation

Install the required Python libraries before running the notebooks:

```bash
pip install numpy pandas matplotlib seaborn scikit-learn jupyter
```

---

## ▶️ How to Run

1. Clone this repository:

```bash
git clone https://github.com/Zvaishali28/-OIBSIP.git
```

2. Open the project folder:

```bash
cd -OIBSIP
```

3. Start Jupyter Notebook:

```bash
jupyter notebook
```

4. Open any task folder.
5. Run the selected notebook from top to bottom.
